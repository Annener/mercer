# context/ — AI-контекст проекта Mercer

Эта папка содержит сжатый контекст для ИИ-ассистентов, работающих над проектом.
Файлы обновляются вручную в конце значимых рабочих сессий.

## Файлы

| Файл | Содержимое |
|------|------------|
| `01-architecture.md` | Сервисная архитектура, схема взаимодействия, порты |
| `02-codebase-map.md` | Карта кодовой базы: модули, роли, ключевые файлы |
| `03-data-models.md` | PostgreSQL-схема (ORM) + Pydantic-контракты |
| `04-pipeline-system.md` | DAG-пайплайны: схема, модели, executor, API |
| `05-current-status.md` | Текущий статус разработки, техдолг, следующие шаги |
| `06-conventions.md` | Соглашения по коду, паттерны, правила |

## Как использовать

В начале сессии с ИИ: вставь содержимое нужных файлов или укажи ссылку.
Для быстрого старта достаточно `01-architecture.md` + `05-current-status.md`.
