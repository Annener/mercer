# Технический долг Mercer

Эта директория содержит каталог задач по устранению технического долга.
Каждый файл описывает **одну** проблему и является отправной точкой для отдельного коммита (и при необходимости — отдельного чата).

## Правила работы с техдолгом

1. **Перед исправлением** — прочитать файл целиком, убедиться в понимании контекста и потенциальных side-effects
2. **Один файл — один коммит** (ветка `techdebt/<имя-файла-без-расширения>`)
3. После исправления — удалить соответствующий `.md` файл из этой директории в том же коммите
4. Если в процессе анализа обнаружена новая проблема — создать новый файл по шаблону
