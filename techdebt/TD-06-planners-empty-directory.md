# TD-06 — Пустая директория app/planners/

**Приоритет:** 🟡 Серьёзный  
**Путь:** `rag-backend/app/planners/`

## Проблема

Директория содержит только `__init__.py` без содержимого. Планировщики
(`Planner`, `LLMRAGPlanner`) находятся в `app/services/planner.py`.

Возможные причины:
1. Остаток незавершённого рефакторинга (планировщики должны были переехать сюда)
2. Исторический артефакт после переноса кода

  или мусор?
- [ ] Проверить импорты: есть ли где-то `from app.planners import ...`?

## Ожидаемое исправление

**Сценарий A (мусор):**  
Удалить директорию `app/planners/` целиком.

**Сценарий B (запланированный переезд):**  
Перенести `Planner` из `app/services/planner.py` в `app/planners/planner.py`,
обновить импорты, создать задачу если нужна поэтапная миграция.
