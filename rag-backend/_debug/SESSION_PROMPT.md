# Session Prompt — копировать в новый чат

> Скопируй блок ниже целиком при старте новой сессии по исправлению багов.

---

```
Репозиторий: https://github.com/Annener/mercer
Проект: Mercer — RAG-чат, FastAPI бэк + JS фронт

Бэкенд — источник правды. Фронт правится под бэк.

Документация проекта:
- Контракты API: rag-backend/_debug/CONTRACTS.md
- Прогресс фикса: rag-backend/_debug/PROGRESS.md

Правила работы:
1. Перед правкой фронта — смотрим CONTRACTS.md для нужного эндпоинта
2. После каждого исправления — обновляем PROGRESS.md (меняем 🔴 на ✅)
3. Один эндпоинт / один фронт-файл за сессию
4. Фронт-файл смотрим через GitHub MCP, правим тоже через него

Текущая задача: [ВСТАВЬ СЮДА — например: "C1 POST /chat/create — проверить frontend/js/chat.js"]

Начни с: найди фронт-файл, который вызывает этот эндпоинт,
сравни с контрактом из CONTRACTS.md, предложи правки.
```

---

## Как использовать

1. Открой новый чат
2. Вставь блок выше
3. В строку `Текущая задача:` подставь ID из PROGRESS.md (например `C1`, `C8`, `S36`)
4. Работаем по одному пункту до статуса ✅
5. Следующая сессия — следующий пункт
